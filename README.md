This is the code for the method introduced in the paper "Rollout-Based Charging Scheduling for Electric Truck Fleets in Large Transportation Networks" submitted to IEEE CDC 2026. 
It includes the comparisons between the proposed approach and the exact optimal solution in small scale systems, and the comparisons with three base policies, in large scale systems.
